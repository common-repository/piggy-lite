== Description ==

Piggy is a mobile sales statistics add-on for the popular **WP e-Commerce, Cart66, WooCommerce & Shopp** plugins. When installed, Piggy allows access to a web-application on your ios or Android smartphone that can be used to view real-time sales and projections for your WordPress e-commerce website.

For more information about Piggy and our other plugins, visit [BraveNewCode](http://www.bravenewcode.com/?utm_campaign=piggy-front&utm_medium=web&utm_source=wordpressdotorg "BraveNewCode").  Also check out our professional mobile plugin for WordPress, [WPtouch](http://www.bravenewcode.com/wptouch/?utm_campaign=piggy-front&utm_medium=web&utm_source=wordpressdotorg "WPtouch").

== Installation ==

= Minimum Requirements =

* Self-hosted WordPress(.org) installation
* WordPress admin account access
* WP E-Commerce 3.7.6+, Shopp 1.1+, or Cart66 1.1.6+, or WooCommerce installed
* PHP 5.2 or higher on your server
* An iPhone, iPod touch (iOS5 or newer) or Android (Android OS3 or newer) smartphone device to access and view Piggy

= Adding Piggy via the WordPress admin panel: =

1. Visit Plugins->Add New in the admin.
2. Enter "Piggy" for the search term, and click the "Search Plugins" button
3. Find the "Piggy" plugin in the list and click "Install Now" to install the plugin

= Viewing Piggy on Your iPhone, iPod touch or Android Device =

Piggy shows you its Web URL on the General tab in Piggy's settings. Visit this URL on your iPhone, iPod touch or Android device and enter the passcode in in Piggy's settings.
